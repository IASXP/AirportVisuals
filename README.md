## Getting Started

Currently this project is only tested on mac using X-Code

### Frameworks

1. libcairo.dylib
2. libgobject.dylib
3. libgtk.dylib

### Includes

1. gtk
2. cairo
3. glib
4. pango
5. harfbuzz
6. gdk-pixbuf
7. atk

```gcc main.c drawing.c parser.c -o nd_display `pkg-config --cflags --libs gtk+-3.0 cairo` && ./nd_display```
